# Погода по городам России — полноценный сайт

Отдельный сайт с **интерактивной картой** России: перемещение, масштаб (приближение/отдаление), маркеры городов с погодой — по аналогии с [Яндекс.Погода — карта](https://yandex.ru/pogoda/ru/sobolevo/maps/temperature).

## Возможности

- **Главная**: интерактивная карта России (Leaflet + OpenStreetMap), маркеры по всем городам, при клике — переход к погоде в городе.
- **Список городов**: сетка городов с текущей температурой.
- **Страница города**: погода, прогноз и интерактивная карта, центрированная на выбранном городе (карту можно двигать и масштабировать).

## Запуск

1. Скопируйте `cities.json` из `../weather_app/cities.json` в эту папку (или запустите `python ../export_cities_json.py` и скопируйте результат сюда).
2. Откройте `index.html` через локальный веб-сервер (из-за CORS при `file://` загрузка `cities.json` может не сработать).

Пример через Python:
```bash
cd weather-site
python -m http.server 8080
```
Откройте в браузере: http://localhost:8080

## Деплой

Залить содержимое папки `weather-site` на любой хостинг (GitHub Pages, Netlify, Vercel и т.д.). Убедитесь, что в корне раздачи есть `index.html`, `styles.css`, `app.js` и `cities.json`.

## Технологии

- Leaflet — интерактивная карта (перемещение, зум, маркеры).
- OpenStreetMap — подложка карты.
- Open-Meteo API — погода.
