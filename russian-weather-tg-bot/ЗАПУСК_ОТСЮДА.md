# Запуск бота погоды и мини-приложений

## 1. Токен бота (обязательно)

Откройте файл **`.env`** в этой папке и вставьте токен от @BotFather:

```
TELEGRAM_TOKEN=123456789:ABCdefGHI...
```

Сохраните файл.

## 2. Запуск бота

В этой папке выполните:

```powershell
python bot.py
```

Или дважды щёлкните **ЗАПУСТИТЬ_БОТА.bat** (если есть).

## 3. Мини-приложения (погода + Pac-Man)

В репозитории настроен деплой на **GitHub Pages**:

- Включите: **GitHub** → репозиторий **russian-weather-tg-bot** → **Settings** → **Pages** → **Source**: **GitHub Actions**.
- После push мини-приложения будут доступны:
  - Сайт погоды: https://gpro7724-netizen.github.io/russian-weather-tg-bot/weather_app/
  - Pac-Man: https://gpro7724-netizen.github.io/russian-weather-tg-bot/mini_app/

В `.env` уже прописаны `WEATHER_APP_URL` и `MINI_APP_URL` для этих адресов. После деплоя кнопки в боте откроют эти приложения.

## 4. Репозиторий

Git remote настроен с вашим токеном для push. Рекомендуется после первого использования отозвать старый токен в GitHub и создать новый (и обновить remote), так как токен был передан в чате.
