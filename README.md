## Бот погоды и новостей по городам‑миллионникам России

**Важно:** если токен от BotFather был выдан для **@Russianweather1_bot** (с цифрой 1), то в Telegram нужно открывать именно его: **https://t.me/Russianweather1_bot**  
Бот **@Russianweather_bot** (без единицы) — это другой бот, ему нужен свой токен.

**Возможности:**
- **Погода** — текущая погода по любому городу‑миллионнику РФ (Open-Meteo, без ключа).
- **Новости** — актуальные новости: по умолчанию RSS (Лента, Ведомости); при наличии ключа NewsAPI — новости по выбранному городу.
- **Города** — Москва, Санкт‑Петербург и все основные города‑миллионники России.

Для работы нужен только **TELEGRAM_TOKEN** в `.env`. Ключи погоды и новостей не обязательны.

---

### 1. Окружение

```powershell
cd "c:\Users\Administrator\Desktop\tg bot"
pip install -r requirements.txt
```

### 2. Настройка

Создайте файл `.env`:

```text
TELEGRAM_TOKEN=ваш_токен_от_BotFather
```

Опционально:

- **NEWS_API_KEY** — новости по городу через NewsAPI.
- **VK_ACCESS_TOKEN** — подтягивать посты новостных групп ВКонтакте (РИА, Лента, РБК и др.). Токен: [vk.com/dev/access_token](https://dev.vk.com/docs/access_token) (права: доступ к стене).

Новости из **Telegram** (каналы СМИ) подмешиваются автоматически через RSS-мосты (RSSHub, tg.i-c-a.su), ключи не нужны.

### 3. Запуск

```powershell
python bot.py
```

Откройте бота в Telegram (например, `@Russianweather1_bot`) и отправьте `/start`.

### 4. Команды

- `/start` — приветствие и выбор города.
- `/city` — снова выбрать город.
- `/help` — справка.

После выбора города бот присылает погоду и блок новостей.
